# Half Adder Layout Design (Sky130 – Magic VLSI)

## Overview
This project presents the transistor-level CMOS layout of a Half Adder designed using the Sky130 PDK in Magic VLSI.

## Design Details
- Implemented basic Half Adder functionality using XOR and AND logic
- Designed at transistor level using CMOS logic
- Layout created following Sky130 design rules

## Tools & Technology
- Magic VLSI
- Sky130 PDK

## Verification
- Performed Design Rule Check (DRC) to ensure layout correctness
- Layout follows basic design constraints and rules

## Files Included
- `layout.mag` – Magic layout file
